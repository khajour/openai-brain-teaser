# Azure OpenAI

## Install

```sh
$ python akstart.py
$ pip install jupyterlab
$ pip install openai --upgrade
$ pip install gTTS --upgrade
$ pip install pynq 
$ pip install azure-cognitiveservices-speech


$ set AZURE_OPENAI_KEY="**********************"
$ set AZURE_OPENAI_ENDPOINT="******************"
$ set AZURE_SPEECH_KEY="******************"



$ jupyter lab
```
